# 🍀 Celtic Girona - Web Oficial

Una web moderna, responsiva i plena de sorpreses per al club esportiu Celtic Girona!

## 📋 Contingut

### Pàgines Principals

1. **Inici** - Portada espectacular amb animacions
   - Hero section amb gradients dinàmics
   - Partícules animades
   - Efectes parallax al scroll

2. **Qui Som** - Presentació del club
   - Historia del club
   - Objectius i missió
   - Comunitat i valors

3. **Premsa & TV** - Notícies i cobertura
   - Articles de premsa
   - Vídeos de televisió
   - Últimes notícies

4. **Botiga Online** - Venda de productes
   - Samarretes oficials
   - Bufandes i roba
   - Accés a botiga externa

5. **Contacte** - Informació i xarxes socials
   - Ubicació i telèfon
   - Formulari de contacte
   - Enlaces a xarxes socials
   - Facebook, Instagram, Twitter, YouTube, TikTok

## 🚀 Com Usar

1. **Obrir la web:**
   - Obrir `index.html` en un navegador

2. **Personalització:**
   - Canviar els colors en `styles.css` (variables CSS)
   - Actualitzar contingut en `index.html`
   - Afegir/modificar funcionalitat en `script.js`

3. **Publicar:**
   - Puja tots els fitxers a un hosting (Netlify, GitHub Pages, etc.)
   - Actualitza els enllaços de xarxes socials
   - Connecta amb la botiga online real

## 🔧 Personalització

### Canviar Colors
En `styles.css`, modifica les variables CSS:
```css
:root {
    --primary: #22c55e;      /* Verd principal */
    --secondary: #16a34a;    /* Verd secundari */
    --dark: #0f172a;         /* Fons fosç */
    --light: #f8fafc;        /* Fons clar */
}
```

### Afegir Imatges
- Crea una carpeta `/images`
- Afegeix les teves fotos
- Actualitza els `src` en l'HTML

### Xarxes Socials
Actualitza els URLs en la secció de contacte:
```html
<a href="https://facebook.com/tuclub" target="_blank">
```
